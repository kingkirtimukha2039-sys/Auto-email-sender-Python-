# Auto-email-sender-Python-
Auto Email Sender (Python) Install: pip install secure-smtplib Setup: Replace SENDER_EMAIL with your Gmail Go to myaccount.google.com → Security → App Passwords → generate one Replace SENDER_PASSWORD with that app password Run: python auto_email_sender.py Features: Send to 1 person, bulk send, personalized emails
# Auto Email Sender 📧

A simple Python script to send single, bulk, and personalized emails using Gmail SMTP.

## Features
- Send email to one person
- Send same email to multiple people
- Send personalized emails

## How to Run
pip install secure-smtplib
python auto_email_sender.py

## Tech Used
- Python
- smtplib
- MIME
